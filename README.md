# EP005-CP--Individualized-Sepsis-Multi-Treatment-Effect-Prediction
This is an Introduction to Deep Learning project focusing on the development of a multi-treatment sepsis effect algorithm with consideration of each patient's data.

## Repository Layout

- `algorithm/sepsis_causal_project/`: model code, configs, scripts, and reports.
- `data/README.md`: expected local data/artifact layout.

Local large datasets and model artifacts are intentionally excluded from git.
