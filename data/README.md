# Data Layout

This folder contains data-only assets for the project.

- `challenge-2019/`: PhysioNet 2019 challenge training data.
- `mimic_demo/`: local MIMIC demo subset.
- `mimic_iv_demo_v2_2/`: MIMIC-IV demo package.
- `raw_datasets/`: raw staging area.
- `tmp_dl/`: temporary download staging area.
- `sepsis_causal_artifacts/`: model outputs (prepared data, checkpoints, eval, tuning).
- `docs/`: dataset notes and references.
